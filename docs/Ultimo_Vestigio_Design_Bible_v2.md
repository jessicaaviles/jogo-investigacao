# Visão do Produto

Último Vestígio é um jogo multiplayer cooperativo onde grupos resolvem
crimes utilizando observação, dedução e uma IA investigativa. A IA
auxilia o raciocínio humano, nunca o substitui.

Objetivos: - Fazer o jogador sentir-se protagonista de uma série de
investigação. - Valorizar observação e colaboração. - Transformar
evidências em narrativa.

# Princípios de Design

1.  Fotografia é protagonista.
2.  Interface é silenciosa.
3.  Cada tela deve parecer um frame de uma série premium.
4.  A tensão vem da descoberta, não do horror.
5.  Todo componente precisa ter propósito.

# Público-alvo

18--45 anos; fãs de investigação, escape rooms, board games modernos e
thrillers policiais.

# Fantasy do Jogador

"Sou um investigador capaz de conectar fatos invisíveis aos demais
usando inteligência e trabalho em equipe."

# Core Loop

Lobby → Matchmaking → Introdução do caso → Cena do crime → Coleta de
evidências → Análise IA → Interrogatórios → Debate → Votação → Resolução
→ Recompensas.

# Direção Artística

Referências: Mindhunter, True Detective, Sherlock, Zodiac, Prisoners,
Dark.

Evitar: - Cyberpunk - Neon - Terror explícito - HUD futurista - Visual
cassino - Interface corporativa

# Paleta Oficial

Fundos: - #1F2A30 Azul petróleo - #2C302E Grafite quente - #3D4336 Verde
oliva - #4A352F Marrom café - #4D2D35 Bordô profundo

Texto: - Off-white - Cinza quente

Ações: - Dourado envelhecido - Verde oliva claro

# Tipografia

Títulos: Serif editorial. Texto: Sans moderna. Botões: Sans SemiBold.
Escala: 12 / 14 / 16 / 20 / 24 / 32 / 48.

# Grid

8pt grid, margens 24px, raio 16px, espaçamento 8/16/24/32.

# Biblioteca de Componentes

Botões: - Primário - Secundário - Ghost - Danger

Cards: - Evidência - Personagem - Caso - Documento - Estatística

Outros: Chips, Tabs, Bottom Navigation, Bottom Sheet, Modal, Timeline,
Toast, Progress Bar, Badge, Avatar, Selo, Carimbo, Documento, Galeria,
Player Card.

# Ícones

Traço 2px, cantos arredondados, estilo minimalista.

# Motion

Fade 250ms, slide 300ms, blur, parallax leve, feedback tátil.

# Fotografia

Lentes 35--50mm, profundidade de campo curta, granulação sutil, luz
fria + tungstênio, chuva, fumaça discreta, composição cinematográfica.

# Universo

Hotel Vesper, Delegacia Central, Laboratório Forense, Arquivo Municipal,
Mansão Blackwood, Floresta de Alder, Porto Industrial.

# Personagens Oficiais

Equipe: - Sofia Mendes (detetive) - Noah Carter (perito) - Maya Costa
(analista IA) - Victor Hale (inspetor)

Cada personagem deve manter aparência, roupas, paleta e personalidade
constantes.

# Caso Oficial --- O Quarto 7

Vítima: Helena Duarte. Local: Hotel Vesper. Suspeitos: gerente, hóspede,
jornalista, segurança. Elementos: chave, câmera, bilhete, relógio
quebrado, digital parcial. Toda geração futura deve utilizar este caso
como padrão.

# UX

Fluxos completos para onboarding, lobby, investigação, IA, votação,
perfil, ranking, configurações e temporada.

# Gamificação

XP, níveis, títulos, conquistas, temporadas, ranking global, coleção de
casos.

# Inteligência Artificial

Funções: - resumir pistas - encontrar padrões - gerar hipóteses -
explicar deduções - apontar inconsistências

# Multiplayer

Salas públicas e privadas, convite, matchmaking, chat, votação secreta,
resultado sincronizado.

# Áudio

Jazz investigativo moderno, ambiência de chuva, papéis, máquinas, câmera
fotográfica e feedbacks discretos.

# Acessibilidade

Alto contraste, tamanho de fonte, legendas, redução de movimento,
daltonismo, haptics.

# Prompt Bible

Toda IA deve obedecer: - mesmo Design Bible - mesmo caso - mesmos
personagens - mesma paleta - mesma tipografia - mesma navegação -
interface discreta - fotografia hiper-realista - atmosfera HBO/Netflix -
nunca neon, terror ou cyberpunk
